# place2be

place2be ist eine App, die Nutzerinnen und Nutzern Empfehlungen für öffentliche Orte gibt, an denen sie eine schöne Zeit verbringen können. 
Im Mittelpunkt stehen frei zugängliche Orte wie Parks, Plätze, Promenaden, Einkaufsmeilen oder andere urbane Treffpunkte, die sich besonders für kostenlose Freizeitaktivitäten eignen.

## Idee

place2be hilft Menschen dabei, spontan schöne öffentliche Orte zu entdecken. Wenn Nutzerinnen und Nutzer einen empfohlenen Ort besuchen, bestätigen sie per Standort, dass sie tatsächlich vor Ort sind. 
Anschließend beantworten sie 2-3 kurze Fragen zum Ort, zum Beispiel zu Sicherheit, Atmosphäre oder Aufenthaltsqualität.

Dieses Feedback beeinflusst das Ranking des Ortes: Positive Bewertungen lassen einen Ort steigen, negative Rückmeldungen können ihn sinken lassen.

## Features
### MVP

Im MVP konzentriert sich place2be auf:

- Empfehlungen für öffentliche, primär kostenlose Orte
- Standortbasierte Bestätigung vor Ort
- Kurze Bewertung über 2-3 Fragen
- Dynamisches Ranking durch echtes Community-Feedback

### Nice to have
- Punkte als Gamification-Anreiz für abgegebene Bewertungen

#### Gamification

Für jede gültige Bewertung erhalten Nutzerinnen und Nutzer Punkte. Dadurch entsteht ein spielerischer Anreiz, Orte zu besuchen, Feedback zu geben und zur Qualität der Empfehlungen beizutragen.

### Ranking-System

Das Ranking eines Ortes basiert auf dem Feedback der Community. Orte können steigen, wenn Besucherinnen und Besucher sie positiv bewerten, oder sinken, wenn Aspekte wie Sicherheit, Atmosphäre, Sauberkeit oder Aufenthaltsqualität schlechter eingeschätzt werden.

So bleibt place2be aktuell und orientiert sich an realen Erfahrungen statt nur an statischen Informationen.

## Ausblick

Nach dem MVP könnte place2be ermöglichen, neue Orte vorzuschlagen. Dafür gibt es zunächst eine Basisauswahl an bekannten Orten in der App.

Wenn Nutzerinnen und Nutzer an einem Ort sind, der noch nicht in der App erfasst ist, können sie diesen markieren. 
Sobald mehrere Personen denselben Ort markiert haben, kann er automatisch oder nach Prüfung in die zentrale Datenbank aufgenommen werden.

## Ziel

place2be soll Menschen helfen, schöne öffentliche Orte zu entdecken, bessere Freizeitentscheidungen zu treffen und gemeinsam eine aktuelle Karte lebenswerter öffentlicher Orte aufzubauen.
